# TLv8 平台-前后端分离版

#### 介绍
基于SpringBoot+Vue开发的后台管理系统，项目集成了工作流引擎、报表引擎和组织机构权限管理后台，可以应用于OA、HR、CRM、PM等系统开发。

#### 软件架构
SpringBoot+Vue

后端：SpringBoot+MyBatis

前端：Ant Design Vue


#### 安装教程

1. 主目录运行：mvn install
2. tlv8-main: 运行TLv8Application
3. 运行前端：

进入tlv8-ui:

npm install

npm run serve
