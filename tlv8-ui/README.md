#### 项目介绍  

 Vue 3 + Ant Design Vue 2 + Axios 前端模板，提供快速开发



#### 项目结构

```
Pear Admin Ant
│
├─dist  打包目录
│
├─src 源码
│  │
│  |-api 网络请求
│  |
│  |-assets 静态资源
│  |
│  |-component 常用组件    
│  |   
│  |-config 配置文件
│  |
│  |-directives 指令目录
│  |
│  |-layout 布局组件
│  |
│  |-mock 模拟数据
│  |
│  |-router 路由管理
│  | 
│  |-utils 工具类
│  |
│  |-view 页面目录
│  |
│  |-App.vue 主体视图
│  │
│  └─main.js 入口文件
│  
└─package.json 配置文件

```


#### 快速上手

```

// 依赖安装

npm install

// 开发预览

npm run serve

// 开发打包

npm run build

// 生产打包

npm run build-prod

// 项目分析

npm run analyzer

```
